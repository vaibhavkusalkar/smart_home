## Installation
Follow the steps properly -

- Flutter SDK Must be installed properly.

- Clone Repository
```bash
  gh repo clone vaibhavkusalkar/smart_home
```

- Get into the Project folder (i.e smart_home)
```bash
  cd smart_home
```

- Run
```bash
  flutter clean
```
```bash
  flutter pub get
```
The App is ready to use !
